## 🐍 1. 주석 사용법 및 기초

- 주석: 코드 옆에 설명하는 글, 코드와 구분을 하기 위해 '#'을 앞에 붙인다.
- 한줄메모는 '#',
- 여러줄 메모는 ''' 또는 """

(ex) 3 #정수 , '안녕' #문자열
     #한줄 메모 
     '''       """
     여러줄     여러줄
     메모       메모
     '''       """
- print(): 파이썬에서 가장 기본이 되는 함수, 데이터를 출력하기 위해 사용되는 함수

🧠 정리
주석을 사용하면 코드의 의도를 명확히 표현할 수 있고,
나중에 코드를 다시 확인할 때 전체 흐름을 빠르게 이해/파악할 수 있다.

## 🐍 2. 변수와 자료형
- 자료형: 문자와 숫자 등 다양한 자료의 종류
  - 숫자형: 숫자로 이루어진 자료형
  - 문자열: 문자로 이루어진 자료형
  -> 문자 앞뒤에 `' '` 또는 `" "` 사용 
  -> 따옴표 세개(`'''` 또는 `"""`)를 사용하면 여러 줄 문자열 작성 가능
리스트(List): 자료 목록, 리스트 안에는 자료형
 - 숫자형, 문자열 등 서로 다른 자료를 함께 저장 가능
- 변수(Variable):데이터를 저장하는 메모리 공간

변수 이름 규칙:
(1) 숫자, 알파벳, 한글, 언더스코어(_) 등은 사용 가능
(2) 첫 글자는 숫자로 시작할 수 없음
(3) 숫자로만 구성된 변수 이름은 불가능
(4) 파이썬 문법에서 사용하는 예약어 (예: for, while, if 등) 는 변수 이름 사용 불가
(5) 빈칸과 연산자(+, -, % 등) 는 변수 이름안에 사용 불가

🧠 정리
변수는 값을 저장하는 공간이며,
자료형에 따라 저장되고 처리되는 방식이 달라진다는 점을 이해했다.

## 🐍 3. 산술 연산

## 🐍 4. 문자열 연산
- `+` : 문자열 더하기
- `*` : 문자열 곱하기

🧠 정리
문자열도 덧셈과 곱셈이 가능하지만, 
뺄셈이나 나눗셈은 불가능하다는 점을 이해했다.

## 🐍 5. 리스트와 슬라이싱
- 인덱스(Index): 원소의 순서 , 원소의 순서는 0부터 시작
- 인덱싱(Indexing): 인덱스를 이용하여 특정 위치의 원소를 불러오는 것
- 기본형: 문자열[인덱스] , 리스트[인덱스]
- 슬라이싱(Slicing): 문자열이나 리스트의 일부분을 잘라서 가져오는 것
- 기본형: 문자열[a:b] , 리스트[a:b]

설명:
- a: 시작 인덱스
- b: 종료 인덱스
종료 인덱스는 항상 가져올 인덱스보다 1 큰 값

🧠 정리
리스트와 문자열은 같은 방식으로 인덱싱과 슬라이싱이 가능하며, 종료 인덱스는 포함되지 않는다는 점을 명확히 이해했다.

## 🐍 6. input()
- input(): 키보드로부터 입력한 데이터를 가져오는 함수
- 기본형: 변수이름 = input()

🧠 정리
input()으로 입력받은 값도 변수에 저장되며,
입력값은 기본적으로 문자열이라는 점,
그리고 변수와 input()의 차이점을 이해했다.

## 🐍 7. 자료형 형 변환
- 형 변환: 어떤 자료형을 다른 자료형으로 바꾸는 것
- 기본형: 형 변환 함수(자료)
- 형 변환 함수: 프로그래밍 및 데이터 베이스에서 특정 자료형(데이터 타입)을 다른 자료형으로 변환하여 데이터의 형식을 변경하는 함수

|  구분  |   이름   |   형 변환 함수   |
|:------:|:--------:|:--------------:|
|  정수  | integer |      int()      |
|  실수  |  float  |    float()      |
| 문자열 |  string  |     str()      | 
| 리스트 |   list   |     list()     |

에러 상황 및 이유 설명
- input(''): 아무것도 입력 안함
- input(' '): 공백
- input(10,000): 쉼표로 인해 정수가 아닌 문자열 형태
- input(20살): 문자 포함
- input(3.5): 소수점 문자열은 int()로 직접 변환 불가

🧠 정리
형 변환 함수의 종류를 구분할 수 있게 되었고,
input() 사용 시 자료형에 따라 에러가 발생할 수 있다는 점을 이해했다.

## 8. 🐍 비교 연산과 논리 연산
- 논리형(Boolean): 참(True) 과 거짓(False) 단 두 가지 값만으로 논리적인 상태를 표현하는 자료형
- 비교 연산: 두 가지 자료를 비교하는 과정
- 비교 연산자: 비교 연산할 때 사용하는 연산자
- 부등식: 두 숫자 또는 두 식의 크기를 비교하는 수식
- 부등호: 부등식에서 사용하는 기호

|  구분  |  기호  |
|:------:|:------:|
|  같다  |   ==   |
| 같지 않다 |  !=  |
| 왼쪽이 더 크다 | > |
|오른쪽이 더 크다 | < |
| 같거나 왼쪽이 다 크다 | >= |
|같거나 오늘쪽이 더 크다 | <= |

- 논리 연산: 두 논리를 연결할 때 사용
- 논리 연산자: 논리 연산 할 때 사용하는 연산자
  종류: and, or, not
- and 연산자: 조건이 모두 True 일때 True

| 과정 | 결과 |
|:----:|:----:|
| True and True |  True  |
| True and False |  False  |
| False and True |  False  |
| False and False |  False  |

- or 연산자: 둘 중 하나 이상이 True 라면 True

| 과정 | 결과 |
|:----:|:----:|
| True or True | True |
| True or False | True |
| False or True | True |
| False or False | False |

- not 연산자: True 는 False 로, False 는 True 로 변환
not True -> False
not False -> True

## 9. 🐍 조건문
- 조건문: 특정 조건의 성립여부에 따라 다른 코드를 실행하게 만드는 명령문
- if문: 특정 조건이 True 인지 False 인지 판단하여, True 일 경우에만 지정된 코드 블록을 실행하는 조건 제어문
